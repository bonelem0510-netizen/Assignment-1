# Assignment-1
This is the class assignment. I have been asked to make an application to help a friend improve their social connections.
The user uses the application by entering the inputs into "Enter Time Period of Day."
From there, they press "CHECK PROMPT."
When the user wants to enter a different input or receives the message "ANSWER NOT ACCEPTED," they can press the "CLEAR" button. 
The user's output will appear in the "RESULT WILL APPEAR HERE." section of the application
They use the application by entering the three times of day: Morning, Afternoon, and Night.
They can also enter the three major meals of the day: Breakfast, Lunch, and Dinner.
Additionally, they can add the prefix of "Good" to the three times of day.
By entering these nine inputs, they will receive prompts on activities to connect socially with others. 
The "Good" prefixes are used when the user is feeling happy, hopeful, inspired, or needs a challenge that pushes them out of their shell.
If the user enters an input outside of the nine inputs, spells the words wrong, or doesn't add capitals for each word input, the user will receive a message stating "ANSWER NOT ACCEPTED." They are then allowed to try again.
